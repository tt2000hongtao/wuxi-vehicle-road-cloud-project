## 3. 点位编辑页面

### 3.1 页面目标

点位编辑页面用于维护单个点位的完整档案，并展示当前点位地图、周边点位、设备安装信息和一点一档资料。

### 3.2 页面结构

```text
------------------------------------------------------+
| 标题区：点位名称 / NodeID / 状态 / 操作按钮          |
+-----------------------------+------------------------+
| 基础信息表单                | 嵌入式地图             |
|                             | 当前点位 + 周边点位    |
+-----------------------------+------------------------+
| 页签：设备信息 / 一点一档 / 单据资料 / 工单 / 日志   |
+------------------------------------------------------+
```

### 3.3 标题区

展示：

- 点位名称
- NodeID
- 行政区域
- 点位类型

操作按钮：

- 保存
- 取消
- 重新转换坐标
- 查看一点一档
- 创建工单
- 返回列表

### 3.4 基础信息表单

字段：

- 序号
- 点位类型
- 行政区域
- NodeID
- CrossID
- 点位英文名称（NodeName）
- 点位名称
- 高德展示经度 GCJ-02
- 高德展示纬度 GCJ-02
- 信号机厂商
- 感知点位类型
- 是否自适应点位
- 是否可变车道路口
- 可变车道数量
- 可变车道方向

校验规则：

- NodeID 必填且唯一。
- 点位名称必填。
- 行政区域必选。
- 点位类型必选。
- 原始经纬度必须为合法数字。
- 原始 CGCS2000 坐标仅用于原始数据记录和审计；业务页面默认只显示 GCJ-02 坐标。

### 3.5 嵌入式地图

展示要求：

- 使用高德 Web JS API/地图 SDK。
- 地图中心为当前点位 GCJ-02 坐标。
- 当前点位高亮显示并展示标签。
- 周边 4-5 个最近点位以普通点位样式展示。
- 自动缩放到同时包含当前点位和周边点位的范围。

地图信息面板：

- 当前点位 GCJ-02 坐标
- 转换方式
- 转换时间
- 原始 CGCS2000 坐标审计入口
- 周边点位列表与距离

周边点位列表字段：

- 距离排名
- NodeID
- 点位名称
- 行政区域
- 点位类型
- 距离米

交互规则：

- 点击周边点位，在地图上打开摘要气泡。
- 摘要气泡提供“跳转编辑页”。
- 若当前点位坐标转换失败，地图区域显示异常提示，并提供重新转换按钮。
