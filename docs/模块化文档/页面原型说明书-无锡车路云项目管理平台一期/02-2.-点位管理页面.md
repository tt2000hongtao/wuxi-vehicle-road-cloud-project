## 2. 点位管理页面

### 2.1 页面目标

点位管理页面用于项目人员以表格、卡片、地图三种方式查看和管理点位。三种视图共享同一套筛选条件。

### 2.2 顶部筛选区

筛选项：

- NodeID
- 点位名称
- 点位英文名称
- 点位类型
- 行政区域
- 信号机厂商
- 感知点位类型
- 是否自适应点位
- 是否可变车道路口

操作：

- 查询
- 重置
- 导入点位表
- 导出当前结果
- 列配置
- 视图切换：表格 / 卡片 / 地图

### 2.3 表格视图

表格列：

- 序号
- 点位类型
- 行政区域
- NodeID
- CrossID
- 点位英文名称
- 点位名称
- 高德展示经度 GCJ-02
- 高德展示纬度 GCJ-02
- 信号机厂商
- 感知点位类型
- 是否自适应点位
- 是否可变车道路口
- 可变车道数量
- 可变车道方向
- 设备记录数
- 已安装数量
- 异常数量
- 更新时间

行操作：

- 编辑
- 查看一点一档
- 查看地图
- 查看设备
- 创建工单
- 删除

交互规则：

- 不分页显示，表格内容区通过右侧滚动条浏览。
- 默认将有异常点位排在表格前面。
- 点击整行或点位信息进入点位编辑页面。
- 点击 `删除` 时必须二次确认；若点位下挂有设备，需先到设备管理删除该点位下设备后，才能删除点位。
- 支持列配置、拖动调整列顺序、拖动调整列宽。
- 行政区域以不同颜色标签显示。
- `NodeID` 缺失或重复的导入记录不得进入正式点位库。
- 设备记录数为 0 的点位显示弱提示，便于核对设备表匹配情况。

### 2.4 卡片视图

卡片内容：

- 点位名称
- NodeID
- 行政区域
- 点位类型
- 信号机厂商
- 感知点位类型
- 是否自适应点位
- 是否可变车道路口
- 设备记录数
- 已安装数量
- 异常数量

卡片底部操作：

- 编辑
- 一点一档
- 地图定位

交互规则：

- 点击卡片主体进入点位编辑页面。
- 有开箱验收问题或未关闭工单时，卡片显示异常标签。
- 可按点位状态或行政区域分组展示。

### 2.5 地图视图

地图组件：

- 高德 Web JS API/地图 SDK
- 点位图层
- 点位聚合
- 左侧筛选结果列表
- 右侧点位摘要抽屉

地图点样式：

- 当前筛选范围内点位按行政区域着色。
- 不同行政区域使用不同颜色标识。
- 坐标转换失败的点位不落图，在左侧异常清单中展示。
- 有异常问题的点位增加红色角标。

点击点位弹窗：

- 点位名称
- NodeID
- 行政区域
- 点位类型
- 信号机厂商
- 感知点位类型
- 是否自适应点位
- 是否可变车道路口
- 设备记录数
- 已安装数量
- 开箱验收问题数
- 编辑按钮
- 一点一档按钮

交互规则：

- 点击弹窗“编辑”进入点位编辑页面。
- 地图缩放或拖拽后，左侧列表联动当前视野点位。
- 地图搜索支持 NodeID 和点位名称。
