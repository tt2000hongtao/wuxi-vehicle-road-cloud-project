## 9. 合同管理页面

### 9.1 合同驾驶舱

页面目标：展示合同金额、流向、签署、履约和风险总览。

```text
+--------------------------------------------------------------------------------+
| 合同管理驾驶舱                              [导入合同测算表] [风险中心]        |
+--------------------------------------------------------------------------------+
| 总合同金额 | 移动流出 | 流回天安 | 天安流出 | 原厂采购 | 毛利 | 待签合同       |
+--------------------------------------------------------------------------------+
| 合同流 Sankey 图                         | 分项金额排行                       |
| 移动 -> 浪潮 -> 天安 -> 大为             | 1.1 信号机网联化...                 |
| 移动 -> 四维 -> 天安 -> 联通 -> 中信科   | 1.8 RSU...                          |
+--------------------------------------------------------------------------------+
| 签署计划预警 | 设备匹配异常 | 数量差异 | 负毛利/价格倒挂                     |
+--------------------------------------------------------------------------------+
```

### 9.2 合同流图谱

页面目标：将 `合同流` sheet 中一级至六级分包商展示为图谱。

筛选项：

- 分项编号
- 项目名称
- 拆分包名称
- 单位名称
- 合同流类型
- 供应商

图谱交互：

- 点击节点查看该单位关联合同、金额、设备和点位。
- 点击边查看上游单位、下游单位、流转金额、价差、关联合同。
- 支持只看“流回天安”的链路。
- 支持只看“天安直接向后流动”的链路。

### 9.5 合同设备匹配页面

页面结构：

```text
+--------------------------------------------------------------------------------+
| 左侧：合同明细                              | 右侧：设备安装项候选             |
| 智能抱杆箱 / 国产定制 / 数量403            | NodeID / 点位 / 安装位置 / 物料     |
| 合同链路：移动->浪潮->天安->大为           | 匹配分数 / 匹配依据 / 确认按钮      |
+--------------------------------------------------------------------------------+
```

匹配依据显示：

- 物料名称相似
- 品牌型号相似
- 合同名称相似
- 供应商一致
- 数量接近
- 部署位置/点位范围一致

### 9.9 设备级资金流页面

页面目标：将前向回款、后向付款、业主回流、质保金等资金状态下沉到设备/合同明细级。

列表字段：

- 合同明细
- 设备安装项
- NodeID
- 资金方向
- 付款方
- 收款方
- 应收/应付金额
- 实收/实付金额
- 计划日期
- 实际日期
- 触发履约事件
- 是否依赖上游回款
- 是否需回流业主车联网集团
- 资金状态

状态规则：

- 上游未回款时，后向付款节点显示 `待上游回款`。
- 天安自有或集成费如果需要回流业主车联网集团，显示 `待回流/部分回流/已回流`。
- 同一合同下不同设备分别显示各自履约节点和资金节点。

- 合同流路径
- 分项编号
- 合同明细名称
- 上游单位
- 下游单位
- 合同金额
- 原厂采购价格
- 毛利
- 签约状态
- 履约状态
- 前向回款状态
- 后向付款状态
- 是否需回流业主车联网集团

点击合同流路径打开链路抽屉，展示从移动/天安到最终供应商的完整流转。
