## 5. 数据导入接口

### 5.1 上传点位管理表并预校验

```text
POST /api/v1/import/sites/preview
```

入口：总览页 `导入点位管理表` 按钮。

模板格式参考：

```text
/Users/tt2000/Documents/天安智联/AI/项目管理工具包/Project Data/点位管理表-无锡车路云.xlsx
```

请求：

- multipart file

返回字段：

- importTaskId
- totalRows
- validRows
- invalidRows
- duplicateNodeIdCount
- coordinateErrorCount
- dictionaryErrorCount
- pendingCompletionRows
- previewRows

### 5.2 确认导入点位管理表

```text
POST /api/v1/import/sites/confirm
```

请求体：

```json
{
  "importTaskId": "xxx"
}
```

返回字段：

- importedRows
- failedRows
- coordinateConvertedRows
- coordinateFailedRows
- pendingCompletionRows

### 5.3 上传设备安装位置表并预校验

```text
POST /api/v1/import/device-items/preview
```

入口：总览页 `导入设备管理表` 按钮。

模板格式参考：

```text
/Users/tt2000/Documents/天安智联/AI/项目管理工具包/Project Data/设备安装位置表.xlsx
```

用途：导入设备点位分布信息，按 NodeID 匹配到点位，并将未匹配记录进入未匹配池。

请求：

- multipart file

返回字段：

- importTaskId
- sheet1TotalRows
- matchedRows
- matchFailedRows
- unmatchedSheetRows
- unpackingIssueRows
- quantityErrorRows
- previewRows

### 5.4 确认导入设备安装位置表

```text
POST /api/v1/import/device-items/confirm
```

请求体：

```json
{
  "importTaskId": "xxx"
}
```

返回字段：

- importedMatchedRows
- importedUnmatchedRows
- createdGovernanceRows
- unpackingIssueRows
