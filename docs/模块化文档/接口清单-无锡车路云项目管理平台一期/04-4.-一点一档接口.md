## 4. 一点一档接口

### 4.1 点位一点一档

```text
GET /api/v1/sites/{siteId}/archive
```

用途：点位编辑页一点一档页签。

返回结构：

```json
{
  "site": {},
  "coordinate": {},
  "neighbors": [],
  "archiveFolder": {},
  "acceptanceForm": {},
  "acceptanceDeviceItems": [],
  "acceptanceTestItems": [],
  "wiringForm": {},
  "wiringItems": [],
  "photos": [],
  "deviceItems": [],
  "contractSummary": {},
  "documents": [],
  "workOrders": [],
  "issues": [],
  "operationLogs": [],
  "statistics": {}
}
```

### 4.2 一路一档目录列表

```text
GET /api/v1/site-archives/folders
```

用途：查看从一路一档资料目录自动扫描出的路口档案文件夹及匹配状态。

请求参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| pageNo | int | 否 | 页码 |
| pageSize | int | 否 | 每页数量 |
| archiveName | string | 否 | 文件夹/档案名称 |
| matchStatus | string | 否 | matched/unmatched/manual_matched/ignored |
| district | string | 否 | 归并后行政区域 |

返回字段：

- id
- siteId
- nodeId
- archiveName
- folderPath
- matchedSiteName
- matchStatus
- matchScore
- archiveCompleteness
- missingItems
- importedAt
- updatedAt

### 4.3 扫描一路一档目录

```text
POST /api/v1/site-archives/scan
```

用途：扫描指定目录，自动生成档案目录、资料文件、验收单、接线表、照片记录。

请求体：

```json
{
  "rootPath": "/Users/tt2000/Documents/天安智联/0000无锡市车路云一体化项目/4.项目管理/深化设计/大为/车路云审核通过点位20251014"
}
```

返回字段：

- scannedFolders
- matchedFolders
- unmatchedFolders
- acceptanceFormsParsed
- wiringFormsParsed
- photosImported
- parseFailedCount

### 4.4 手工匹配档案目录到点位

```text
POST /api/v1/site-archives/folders/{folderId}/link-site
```

请求体：

```json
{
  "siteId": 1001,
  "nodeId": "10372",
  "remark": "人工确认点位名称匹配"
}
```

处理规则：

- 更新 `site_archive_folder.match_status=manual_matched`。
- 回写验收单、接线表、照片的 `site_id`。
- 重新计算一点一档完整度。

### 4.5 验收单详情

```text
GET /api/v1/site-archives/folders/{folderId}/acceptance-form
```

返回字段：

- acceptanceForm
- acceptanceDeviceItems
- acceptanceTestItems
- sourceFile

### 4.6 接线表详情

```text
GET /api/v1/site-archives/folders/{folderId}/wiring-form
```

返回字段：

- wiringForm
- wiringItems
- sourceFile

### 4.7 档案照片列表

```text
GET /api/v1/site-archives/folders/{folderId}/photos
```

返回字段：

- id
- fileId
- fileName
- photoType
- shootingDirection
- isCover
- previewUrl

### 4.8 导出一点一档

```text
POST /api/v1/sites/{siteId}/archive/export
```

请求参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| format | string | 是 | pdf/docx |

返回字段：

- taskId
- downloadUrl
- status
