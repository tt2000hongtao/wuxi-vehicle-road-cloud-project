## 6. 未匹配设备治理接口

### 6.1 未匹配设备查询

```text
GET /api/v1/device-unmatched-items
```

请求参数：

- pageNo
- pageSize
- supplierName
- installPosition
- contractSection
- materialName
- contractName
- governanceStatus

说明：一期初始化展示 `未匹配` 工作表中的 2447 条记录，支持按物料、供应商、合同板块治理到点位、仓库、机房或服务项。

返回字段：

- id
- sourceItemId
- installPosition
- supplierName
- materialCode
- materialName
- contractName
- brandModel
- unit
- contractQuantity
- deliveredQuantity
- inboundQuantity
- requisitionQuantity
- installedQuantity
- warehouseQuantity
- governanceStatus

### 6.2 关联点位

```text
POST /api/v1/device-unmatched-items/{id}/link-site
```

请求体：

```json
{
  "siteId": 1001,
  "nodeId": "10372",
  "remark": "人工匹配到太湖大道北~友谊南路西"
}
```

处理规则：

- 更新未匹配治理状态为 `已关联点位`。
- 更新来源设备安装项 `site_id`、`node_id`、`match_status=matched`。
- 重新计算对应点位设备统计。

### 6.3 标记为非点位资产

```text
POST /api/v1/device-unmatched-items/{id}/mark
```

请求体：

```json
{
  "targetType": "server_room",
  "targetId": 1,
  "governanceStatus": "已关联机房",
  "remark": "交警支队机房设备"
}
```

targetType 可选：

- server_room
- warehouse
- vehicle
- service
- ignored

### 6.4 批量治理

```text
POST /api/v1/device-unmatched-items/batch-governance
```

请求体：

```json
{
  "ids": [1, 2, 3],
  "targetType": "service",
  "governanceStatus": "软件服务项",
  "remark": "合同服务类条目，不纳入点位设备统计"
}
```
