## 3. 点位设备接口

### 3.1 点位设备清单

```text
GET /api/v1/sites/{siteId}/device-items
```

用途：点位编辑页设备信息页签。

请求参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| groupBy | string | 否 | installPosition/supplier/material |

返回字段：

- id
- nodeId
- siteName
- installPosition
- supplierName
- materialCode
- materialName
- contractName
- brandModel
- unit
- contractQuantity
- deliveredQuantity
- inboundQuantity
- unpackingQuantity
- requisitionQuantity
- installedQuantity
- warehouseQuantity
- deliveryOrderIndex
- inboundOrderIndex
- unpackingOrderIndex
- requisitionOrderIndex
- unpackingIssue

### 3.2 设备安装项详情

```text
GET /api/v1/device-items/{itemId}
```

用途：查看单条设备安装项详情。

返回字段：

- 设备安装位置表全字段
- documents
- relatedSite

### 3.3 已匹配设备分页查询

```text
GET /api/v1/device-items
```

用途：设备管理已匹配设备列表。

请求参数：

- pageNo
- pageSize
- nodeId
- siteName
- installPosition
- supplierName
- materialCode
- materialName
- contractName
- hasUnpackingIssue
- isOnline
- isStatisticsCompleted

返回字段：

- id
- nodeId
- siteName
- installPosition
- supplierName
- materialCode
- materialName
- contractName
- brandModel
- unit
- contractQuantity
- deliveredQuantity
- inboundQuantity
- requisitionQuantity
- installedQuantity
- warehouseQuantity
- unpackingIssue
- matchStatus
