## 10. 错误码建议

| 错误码 | 含义 |
|---|---|
| SITE_NODE_ID_DUPLICATED | NodeID 重复 |
| SITE_NOT_FOUND | 点位不存在 |
| COORDINATE_INVALID | 坐标格式错误 |
| COORDINATE_CONVERT_FAILED | 坐标转换失败 |
| DEVICE_ITEM_NOT_FOUND | 设备安装项不存在 |
| DEVICE_ITEM_ALREADY_MATCHED | 设备安装项已匹配 |
| IMPORT_FILE_INVALID | 导入文件格式错误 |
| IMPORT_FIELD_MISSING | 导入字段缺失 |
| UNMATCHED_ITEM_NOT_FOUND | 未匹配设备不存在 |
| SITE_HAS_DEVICE_ITEMS | 点位下挂有设备，需先删除或迁移设备后才能删除点位 |
