## 2. 点位表 `site`

| 字段名 | 中文名 | 类型 | 必填 | 来源 | 说明 |
|---|---|---:|---|---|---|
| id | 系统主键 | bigint | 是 | 系统生成 | 内部主键 |
| serial_no | 序号 | int | 否 | 点位管理表 | Excel 序号 |
| site_type | 点位类型 | varchar(50) | 是 | 点位管理表 | 如十、T、路段、匝道汇入、V2P、环岛、Y |
| district | 行政区域 | varchar(50) | 是 | 点位管理表 | 锡山、新吴、惠山、梁溪、滨湖。经开区按项目最新口径并入新吴区 |
| original_district | 原始行政区域 | varchar(50) | 否 | 点位管理表 | 保留导入原值，如经开 |
| district_normalize_rule | 行政区域归并规则 | varchar(100) | 否 | 系统生成 | 如 `经开->新吴` |
| node_id | NodeID | varchar(50) | 是 | 点位管理表 | 点位唯一业务主键 |
| cross_id | CrossID | varchar(50) | 否 | 点位管理表 | 路口辅助标识 |
| node_name_en | 点位英文名称 | varchar(200) | 否 | 点位管理表 | 原字段：路口英文名称（NodeName），业务页面显示为点位英文名称 |
| site_name | 点位名称 | varchar(200) | 是 | 点位管理表 | 中文点位名称 |
| signal_vendor | 信号机厂商 | varchar(100) | 否 | 点位管理表 | 如华通等 |
| perception_site_type | 感知点位类型 | varchar(100) | 否 | 点位管理表 | 信号机厂商后展示，按导入值标准化为雷视融合、视频、毫米波雷达、RSU、无等 |
| is_adaptive_intersection | 是否自适应点位 | boolean | 否 | 点位管理表 | 建议导入时标准化 |
| is_variable_lane_site | 是否可变车道路口 | boolean | 否 | 点位管理表 | 由可变车道数量、方向或导入字段标准化生成 |
| in_current_signal_scope | 本期信号系统改造范围 | boolean | 否 | 点位管理表 | 原始导入字段，仅用于审计和历史追溯，不在点位页面默认展示或筛选 |
| variable_lane_count | 可变车道数量 | int | 否 | 点位管理表 | 空值表示无或未知 |
| variable_lane_direction | 可变车道方向 | varchar(100) | 否 | 点位管理表 | 可多方向描述 |
| longitude_cgcs2000 | 原始规划经度 CGCS2000 | decimal(12,8) | 是 | 点位管理表 | 原始坐标，不覆盖，仅用于原始数据记录和审计 |
| latitude_cgcs2000 | 原始规划纬度 CGCS2000 | decimal(12,8) | 是 | 点位管理表 | 原始坐标，不覆盖，仅用于原始数据记录和审计 |
| longitude_gcj02 | 高德展示经度 GCJ-02 | decimal(12,8) | 是 | 转换生成 | 业务页面和地图默认展示坐标 |
| latitude_gcj02 | 高德展示纬度 GCJ-02 | decimal(12,8) | 是 | 转换生成 | 业务页面和地图默认展示坐标 |
| coordinate_convert_method | 坐标转换方式 | varchar(50) | 是 | 系统生成 | 如 cgcs2000_as_wgs84_to_gcj02 |
| coordinate_convert_status | 坐标转换状态 | varchar(20) | 是 | 系统生成 | success/failed/pending |
| coordinate_convert_time | 坐标转换时间 | datetime | 否 | 系统生成 | 最近转换时间 |
| site_status | 点位状态 | varchar(50) | 是 | 系统/人工 | 待规划、已规划、已勘察、已设计、待施工、施工中、已安装、已调试、已验收、运维中 |
| device_record_count | 设备记录数 | int | 否 | 自动计算 | 当前点位 matched 设备安装记录行数，由 `device_installation_item` 自动汇总 |
| device_type_count | 设备类型数 | int | 否 | 自动计算 | 当前点位按设备类型去重后的数量，用于卡片和点位详情摘要 |
| installed_quantity | 已安装数量 | decimal(12,2) | 否 | 自动计算 | 当前点位 `installed_quantity` 自动汇总 |
| issue_count | 异常数量 | int | 否 | 自动计算 | 验收问题、坐标异常、未关闭工单、整改问题自动汇总 |
| deleted_at | 删除时间 | datetime | 否 | 系统生成 | 逻辑删除时间 |
| deleted_by | 删除人 | bigint | 否 | 系统生成 | 逻辑删除操作人 |
| delete_reason | 删除原因 | varchar(500) | 否 | 人工输入 | 删除点位时填写；下挂设备未清理时禁止删除 |
| created_at | 创建时间 | datetime | 是 | 系统生成 |  |
| updated_at | 更新时间 | datetime | 是 | 系统生成 |  |

唯一约束：

- `node_id` 唯一。
