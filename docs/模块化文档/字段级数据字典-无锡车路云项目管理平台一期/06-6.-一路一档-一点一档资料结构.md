## 6. 一路一档/一点一档资料结构

本节参考 `/Users/tt2000/Documents/天安智联/0000无锡市车路云一体化项目/4.项目管理/深化设计/大为/车路云审核通过点位20251014` 目录下样例资料重新设计。样例目录按路口建文件夹，每个路口通常包含：

- `验收单.docx`
- `接线表.xls`
- `点位照片.jpg`

系统应支持从该类目录自动归集资料，自动识别路口文件夹、验收单、接线表、照片，并尽量解析结构化字段。

### 6.1 路口档案目录表 `site_archive_folder`

| 字段名 | 中文名 | 类型 | 必填 | 来源 | 说明 |
|---|---|---:|---|---|---|
| id | 系统主键 | bigint | 是 | 系统生成 |  |
| site_id | 点位 ID | bigint | 否 | 系统匹配 | 匹配成功后关联 `site.id` |
| node_id | NodeID | varchar(50) | 否 | 系统匹配/人工补充 | 目录或验收单中通常不直接出现 NodeID，需通过点位名称匹配 |
| archive_name | 档案名称 | varchar(200) | 是 | 文件夹名称 | 如塘南路－向阳南路口 |
| folder_path | 文件夹路径 | varchar(1000) | 是 | 文件系统 | 原始资料目录路径 |
| matched_site_name | 匹配点位名称 | varchar(200) | 否 | 系统匹配 | 匹配到的点位名称 |
| match_status | 匹配状态 | varchar(30) | 是 | 系统生成 | matched/unmatched/manual_matched/ignored |
| match_score | 匹配分数 | decimal(5,2) | 否 | 系统生成 | 点位名称相似度 |
| archive_completeness | 档案完整度 | decimal(5,2) | 否 | 自动计算 | 按验收单、接线表、照片等资料完整性计算 |
| missing_items | 缺失资料项 | varchar(500) | 否 | 自动计算 | 如缺少验收单、接线表、点位照片 |
| imported_at | 导入时间 | datetime | 是 | 系统生成 |  |
| updated_at | 更新时间 | datetime | 是 | 系统生成 |  |
