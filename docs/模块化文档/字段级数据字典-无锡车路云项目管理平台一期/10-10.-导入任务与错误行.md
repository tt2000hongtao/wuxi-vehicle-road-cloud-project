## 10. 导入任务与错误行

### 10.1 导入任务表 `import_task`

用于承载点位、设备、合同等 Excel 导入的预览、校验、确认和报告，不允许仅依赖前端临时状态。

| 字段名 | 中文名 | 类型 | 必填 | 来源 | 说明 |
|---|---|---:|---|---|---|
| id | 系统主键 | bigint | 是 | 系统生成 |  |
| import_task_id | 导入任务编号 | varchar(64) | 是 | 系统生成 | 接口返回给前端确认导入使用 |
| import_type | 导入类型 | varchar(50) | 是 | 系统生成 | site/device_item/contract/document |
| source_file_name | 原始文件名 | varchar(300) | 是 | 上传文件 | 如点位管理表、设备安装位置表 |
| source_file_path | 文件存储路径 | varchar(1000) | 否 | 系统生成 | 后端文件存储位置 |
| template_matched | 模板是否匹配 | boolean | 是 | 系统生成 | 字段识别是否满足模板要求 |
| total_rows | 总行数 | int | 是 | 系统生成 | Excel 可识别数据行数 |
| valid_rows | 有效行数 | int | 是 | 系统生成 | 可导入正式库的数据行数 |
| invalid_rows | 失败行数 | int | 是 | 系统生成 | 校验失败或待补全行数 |
| matched_rows | 匹配成功行数 | int | 否 | 系统生成 | 设备导入按 NodeID 匹配成功数 |
| unmatched_rows | 未匹配行数 | int | 否 | 系统生成 | 未匹配工作表行数 + Sheet1 匹配失败行数 |
| warning_rows | 警告行数 | int | 否 | 系统生成 | 字典异常、坐标异常等可确认导入的警告行 |
| status | 任务状态 | varchar(30) | 是 | 系统生成 | previewed/confirmed/imported/failed/cancelled |
| report_json | 导入报告 | json | 否 | 系统生成 | 字段映射、错误行摘要、统计结果 |
| created_by | 创建人 | bigint | 是 | 系统生成 |  |
| created_at | 创建时间 | datetime | 是 | 系统生成 |  |
| confirmed_by | 确认人 | bigint | 否 | 系统生成 |  |
| confirmed_at | 确认时间 | datetime | 否 | 系统生成 |  |

### 10.2 导入错误行表 `import_error_row`

| 字段名 | 中文名 | 类型 | 必填 | 来源 | 说明 |
|---|---|---:|---|---|---|
| id | 系统主键 | bigint | 是 | 系统生成 |  |
| import_task_id | 导入任务编号 | varchar(64) | 是 | 系统生成 | 关联 `import_task.import_task_id` |
| sheet_name | 工作表 | varchar(100) | 否 | Excel | Sheet1、未匹配、点位管理表等 |
| row_number | Excel 行号 | int | 否 | Excel | 便于用户回查 |
| error_level | 错误级别 | varchar(20) | 是 | 系统生成 | error/warning |
| error_code | 错误码 | varchar(50) | 是 | 系统生成 | 如 NODE_ID_EMPTY、COORDINATE_INVALID |
| error_message | 错误说明 | varchar(500) | 是 | 系统生成 | 展示给业务用户 |
| raw_row_json | 原始行数据 | json | 否 | 系统生成 | 保留导入行快照 |
