## 4. 设备未匹配池 `device_unmatched_item`

未匹配池可复用 `device_installation_item` 字段结构，但必须额外维护治理字段。

| 字段名 | 中文名 | 类型 | 必填 | 说明 |
|---|---|---:|---|---|
| id | 系统主键 | bigint | 是 |  |
| source_item_id | 来源设备安装记录 | bigint | 否 | 对应导入原始记录 |
| governance_status | 治理状态 | varchar(30) | 是 | 待治理、已关联点位、已关联机房、已关联仓库、车端设备、软件服务项、已忽略 |
| target_type | 关联目标类型 | varchar(30) | 否 | site/warehouse/server_room/vehicle/service/ignored |
| target_id | 关联目标 ID | bigint | 否 |  |
| target_node_id | 关联 NodeID | varchar(50) | 否 | 人工补充 |
| governance_remark | 治理备注 | text | 否 |  |
| governed_by | 治理人 | bigint | 否 |  |
| governed_at | 治理时间 | datetime | 否 |  |
