## 9. 资料文件表 `document_file`

| 字段名 | 中文名 | 类型 | 必填 | 说明 |
|---|---|---:|---|---|
| id | 系统主键 | bigint | 是 |  |
| file_name | 文件名称 | varchar(300) | 是 |  |
| file_type | 文件类型 | varchar(50) | 否 | pdf/xlsx/docx/jpg/png 等 |
| file_category | 文件分类 | varchar(50) | 是 | 勘察、设计、施工、设备、合同、验收、运维、竣工 |
| object_type | 关联对象类型 | varchar(50) | 是 | site/device/contract/inbound/outbound/work_order |
| object_id | 关联对象 ID | bigint | 是 |  |
| object_code | 关联对象编号 | varchar(100) | 否 | 如 NodeID、工单号 |
| storage_path | 存储路径 | varchar(500) | 是 |  |
| version_no | 版本号 | varchar(20) | 否 |  |
| uploaded_by | 上传人 | bigint | 是 |  |
| uploaded_at | 上传时间 | datetime | 是 |  |
| remark | 备注 | text | 否 |  |
