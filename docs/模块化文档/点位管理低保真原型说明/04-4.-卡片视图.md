## 4. 卡片视图

### 4.1 页面草图

```text
+--------------------------------------------------------------------------------+
| [表格] [卡片] [地图]                                  排序：更新时间 / 设备数   |
+--------------------------------------------------------------------------------+
| +----------------------------+ +----------------------------+ +----------------+ |
| | 太湖大道北~友谊南路西      | | 新华路东~锡山大道北        | | ...            | |
| | NodeID: 10372              | | NodeID: 10554              | |                | |
| | 锡山 / 匝道汇入 / 华通     | | 锡山 / 匝道汇入 / -        | |                | |
| | 设备 3  已安装 3  异常 0   | | 设备 2  已安装 1  异常 1   | |                | |
| | [编辑] [一点一档] [地图]   | | [编辑] [一点一档] [地图]   | |                | |
| +----------------------------+ +----------------------------+ +----------------+ |
+--------------------------------------------------------------------------------+
```

### 4.2 卡片字段

- 点位名称
- NodeID
- 行政区域
- 点位类型
- 信号机厂商
- 感知点位类型
- 是否自适应点位
- 是否可变车道路口
- 设备记录数
- 已安装数量
- 异常数量

### 4.3 交互规则

- 点击卡片主体进入点位编辑页。
- 点击 `一点一档` 打开点位编辑页的一点一档页签。
- 点击 `地图` 切换到地图视图并定位该点位。
- 存在异常时卡片顶部显示红色异常条。
