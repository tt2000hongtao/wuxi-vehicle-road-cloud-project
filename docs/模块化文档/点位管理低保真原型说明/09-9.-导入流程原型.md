## 9. 导入流程原型

### 9.1 点位表导入

```text
上传 Excel → 识别字段 → 校验 NodeID/字典/坐标 → 坐标转换 → 预览结果 → 确认导入 → 导入报告
```

导入报告展示：

- 总行数
- 成功行数
- 失败行数
- 重复 NodeID
- 坐标异常
- 字典异常

### 9.2 设备安装表导入

```text
上传 Excel → 识别 Sheet1/未匹配 → 按 NodeID 匹配点位 → 校验数量字段 → 保存单据索引 → 生成未匹配池 → 导入报告
```

导入报告展示：

- Sheet1 总行数
- 匹配成功行数
- 匹配失败行数
- 未匹配工作表行数
- 开箱验收问题数
- 数量异常数
