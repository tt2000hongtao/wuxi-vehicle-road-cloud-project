### 5.2 地图点位样式

| 状态 | 样式 |
|---|---|
| 普通点位 | 蓝色点 |
| 当前选中点位 | 高亮大点 + 标签 |
| 有异常点位 | 红色角标 |
| 坐标异常点位 | 不落图，进入异常列表 |
| 不同行政区域 | 使用不同颜色点位标识 |
| 聚合点 | 显示数量 |

### 5.3 点位弹窗

弹窗内容：

- 点位名称
- NodeID
- 行政区域
- 点位类型
- 信号机厂商
- 感知点位类型
- 是否自适应点位
- 是否可变车道路口
- 设备记录数
- 已安装数量
- 异常数量

弹窗操作：

- 编辑
- 一点一档
- 查看设备

### 5.4 交互规则

- 地图点位使用 GCJ-02 坐标。
- 显示点位坐标全部以 GCJ-02 坐标为准。
- 原始 CGCS2000 坐标仅用于原始数据记录和审计，不在业务页面默认展示。
- 点击地图点位打开弹窗。
- 点击弹窗 `编辑` 进入点位编辑页。
- 地图拖动或缩放后，左侧列表刷新为当前视野点位。
