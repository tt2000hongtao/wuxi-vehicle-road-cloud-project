## 8. 点位编辑页：一点一档页签

### 8.1 页面草图

```text
一点一档                                [导出 PDF] [导出 Word] [打印]
+--------------------------------------------------------------------------------+
| 1. 基础档案                                                                    |
|    NodeID / CrossID / 点位名称 / 行政区域 / 点位类型 / 信号机厂商              |
| 2. 坐标与地图：GCJ-02 坐标、当前点位地图、周边点位距离                         |
| 3. 信号系统信息：信号机厂商、控制方式、相关资料                                |
| 4. 感知与自适应信息：感知点位类型、是否自适应点位                              |
| 5. 可变车道信息：是否可变车道路口、数量、方向                                  |
| 6. 设备安装清单：按设备类型逐项列出安装位置、供应商、物料、数量、状态          |
| 7. 合同与供应商：后项合同、供应商、合同板块、合同金额                          |
| 8. 送货/入库/领料/开箱验收单据：单据编号、日期、附件索引、问题                 |
| 9. 施工与验收资料：施工照片、验收记录、一路一档资料                            |
| 10. 运维工单：工单状态、问题描述、闭环结果                                     |
| 11. 操作日志：导入、编辑、匹配、删除等审计记录                                 |
+--------------------------------------------------------------------------------+
```

### 8.2 导出内容

导出的一点一档应包含：

- 点位基础字段
- GCJ-02 坐标；原始 CGCS2000 坐标仅进入审计记录
- 地图截图或地图链接
- 周边点位列表
- 设备安装清单
- 单据索引与附件
- 异常问题
- 工单与运维记录
