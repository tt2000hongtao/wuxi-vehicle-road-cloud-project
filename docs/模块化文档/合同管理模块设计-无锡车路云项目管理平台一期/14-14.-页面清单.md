## 14. 页面清单

| 页面 | 说明 |
|---|---|
| 合同驾驶舱 | 查看合同金额、流向、履约、风险 |
| 合同流图谱 | Sankey/树图展示多级合同流 |
| 合同流列表 | 导入和维护合同流记录 |
| 合同台账 | 正式合同、计划合同、待签合同 |
| 分项包管理 | 管理 1.1、1.2 等分项包 |
| 设备级合同明细 | 查看硬件、软件、服务明细 |
| 合同设备匹配 | 自动/人工匹配合同明细与设备安装项 |
| 合同详情 | 基础信息、链路、明细、设备、点位、付款、附件 |
| 供应商详情 | 查看供应商合同、设备、点位、金额 |
| 合同风险中心 | 查看金额、数量、签署、匹配等异常 |
| 设备级资金流 | 查看每台设备或每条明细的应收、实收、应付、实付 |
| 前向销售合同 | 管理天安销售合同及上游回款 |
| 后向采购合同 | 管理天安采购合同及背靠背付款 |
