## 14. 页面清单

| 页面 | 说明 |
|---|---|
| 合同关系重构工作台 | 总览宏观流、Word 合同、候选关系、确认状态和审计付款闸门 |
| 宏观合同流骨架 | 查看 Excel `合同流` 尾部资金骨架，支持筛选合同文本和候选关系 |
| Word 合同文本边界 | 复核前向/后向合同方向、主体、金额、税率和条款 |
| 前后向候选关系确认 | 查看“前向合同 → 天安 → 后向合同”候选关系、证据、确认/否决状态 |
| 清单池与设备级拆分 | 将 Excel 清单池行拆到设备/软件/服务项，不把 sheet 当合同 |
| 设备级资金流单元 | 查看前向金额、后向成本、税率、毛利、NodeID、审计和付款锁定 |
| 业主设备级审计 | 记录申报数量/金额、审计确认数量/金额、核减和证据文档 |
| 阶段付款释放 | 管理预付款、到货、安装、验收、审计、质保阶段的可回款/可付款金额 |
| 合同风险中心 | 查看税率不一致、未审计付款、超阶段付款、重复认领等风险 |
| 点位合同穿透 | 从点位/设备查看已确认设备级资金流和阶段付款状态 |
