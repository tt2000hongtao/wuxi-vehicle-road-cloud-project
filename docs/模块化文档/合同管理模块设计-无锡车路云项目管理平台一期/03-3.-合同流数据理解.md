## 3. 合同流数据理解

`合同流` sheet 的主要字段包括：

| 字段 | 含义 |
|---|---|
| 序号 | 分项编号，如 1.1、1.8、1.9.1 |
| 项目名称 | 分项或拆分项名称 |
| 分项小计 | 对应分项金额 |
| 投标价（分项） | 对外投标或上游流转金额 |
| 拆分包名称 | 合同拆包或实际流转包名称 |
| 拆分包价格 | 拆分后的价格 |
| 一级分包商至六级分包商 | 合同链路节点 |
| 一级分包价格至六级分包价格 | 每一级流转价格 |
| 原厂价格分项小计 | 原厂/最终供应价格 |
| 毛利 | 分项或链路毛利 |
| 折扣率 | 毛利或折扣比例 |
| 供应商 | 主要供应商 |
| 备注 | 对应分项明细 sheet |

示例链路：

```text
移动 → 浪潮 → 天安 → 大为
移动 → 四维图新 → 天安 → 联通 → 中信科
移动 → 上研 → 工业安装 → 天安 → 华通
天安 → 华通
天安 → 隆顺
移动 → 上电科
移动自有
```
