## 9. 设备级资金流单元

设备级资金流单元是合同管理的最小核算对象。

```text
前向合同明细
→ 后向合同明细
→ 清单池行
→ NodeID / 点位 / 安装位置
→ 业主设备级审计
→ 上游阶段回款
→ 后向阶段付款释放
```

### 9.1 关键字段

- 宏观合同流 ID。
- 前向合同 ID。
- 后向合同 ID。
- 清单池行 ID。
- 设备/服务名称。
- 品牌型号。
- 数量。
- NodeID。
- 点位名称。
- 前向含税金额。
- 前向税率。
- 后向含税金额。
- 后向税率。
- 税率匹配状态。
- 毛利。
- 业主审计确认数量。
- 业主审计确认金额。
- 当前可回款金额。
- 当前可付款金额。
- 超付风险状态。
- 锁定原因。

### 9.2 审计与付款约束

未绑定 NodeID、未形成设备级审计结果、未满足阶段付款条件或上游资金池不可用时，后向付款必须保持锁定。

### 9.3 税率规则

前向税率、后向税率和发票税率必须作为校验字段。税率不一致时，应进入风险中心，而不是隐藏在备注中。
