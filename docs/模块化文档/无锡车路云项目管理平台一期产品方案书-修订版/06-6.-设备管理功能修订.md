## 6. 设备管理功能修订

### 6.1 设备数据来源

设备管理一期以 `设备安装位置表.xlsx` 为基础数据源。`Sheet1` 中有 NodeID 的数据进入点位设备清单；`未匹配` 工作表进入待治理设备池。

### 6.2 设备台账粒度

设备台账建议分为两层，并在点位详情中按设备类型逐个展示：

1. 设备安装汇总项：来自 Excel 的每一行安装位置记录。
2. 设备资产实例：当存在 SN、二维码或实物资产编号时，拆分成单台设备实例。

若 Excel 中只有数量，没有 SN，则先按安装汇总项管理；后续入库扫码或序列号验证后再拆分为资产实例。

### 6.3 设备与点位关联

关联规则：

- 主关联键：`NodeID`
- 辅助匹配：`点位名称`
- 设备落位：`安装位置`
- 设备识别：`物料编码 + 物料名称 + 品牌型号`

若设备记录无 NodeID，则进入未匹配池，支持人工选择：

- 关联到点位
- 关联到机房
- 关联到仓库
- 标记为车端设备
- 标记为软件/服务类合同项
- 标记为不纳入点位设备统计

### 6.4 设备状态

设备状态由数量字段和单据字段综合判断：

- 合同内：存在后项合同数量
- 已送货：已送货数量大于 0
- 已入库：入库单数量大于 0
- 已开箱验收：开箱验收单数量大于 0
- 已领料：本期领料数量大于 0
- 已安装：已安装数量大于 0
- 在仓库：在仓库数量大于 0
- 验收异常：开箱验收问题非空
