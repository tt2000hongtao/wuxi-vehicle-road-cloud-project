# 点位管理低保真原型说明

## 1. 页面范围

本原型说明覆盖点位管理一期核心页面：

- 点位管理首页
- 表格视图
- 卡片视图
- 地图视图
- 点位编辑页
- 设备信息页签
- 一点一档页签
- 导入与异常处理入口

## 2. 点位管理首页

### 2.1 页面布局

```text
+--------------------------------------------------------------------------------+
| 顶部标题：点位管理                                                             |
| 操作按钮：导入点位管理表 / 导入设备管理表 / 导出 / 坐标异常 / 未匹配设备       |
+--------------------------------------------------------------------------------+

```

### 2.2 顶部操作

| 操作 | 说明 |
|---|---|
| 导入点位管理表 | 上传点位管理表，生成点位库；模板格式参考 `/Users/tt2000/Documents/天安智联/AI/项目管理工具包/Project Data/点位管理表-无锡车路云.xlsx` |
| 导入设备管理表 | 上传设备点位分布信息，按 NodeID 匹配设备；模板格式参考 `/Users/tt2000/Documents/天安智联/AI/项目管理工具包/Project Data/设备安装位置表.xlsx` |
| 导出 | 导出当前筛选结果 |

| 未匹配设备 | 进入设备未匹配治理页面 |

### 2.3 筛选项

| 筛选项 | 控件 | 说明 |
|---|---|---|
| NodeID | 输入框 | 支持精确/模糊搜索 |
| 点位名称 | 输入框 | 支持中文名称搜索 |
| 行政区域 | 下拉多选 | 锡山、新吴、惠山、梁溪、滨湖；经开区在项目过程中已并入新吴区，导入原值仅保留为审计字段 |
| 点位类型 | 下拉多选 | 十、T、路段、匝道汇入、匝道汇出、V2P、环岛、Y |
| 信号机厂商 | 下拉多选 | 来自字典或导入数据 |
| 感知点位类型 | 下拉 | 全部、雷视融合、视频、毫米波雷达、RSU、无 |
| 是否自适应点位 | 下拉 | 全部、是、否 |
| 是否可变车道路口 | 下拉 | 全部、是、否 |

### 3.3 交互规则

- 默认将有异常点位排在表格前面。
- 点击点位信息或 `编辑` 进入点位编辑页。
- 点击 `删除` 时必须二次确认；若该点位下挂有设备，需先到设备管理删除该点位下设备后，才能删除点位。
- 点击 `设备记录数` 跳转到点位编辑页的设备信息页签。
- 点击 `异常数量` 展示异常弹窗，包含开箱验收问题、坐标异常、未关闭工单。
- 支持列配置，用户可隐藏不常用字段、拖动调整列顺序、拖动调整列宽。
- 行政区域以不同颜色标签显示。
- 支持批量导出当前筛选结果。

### 6.2 基础信息区

字段分组：

基础标识：

- 序号
- NodeID
- CrossID
- 点位名称
- 点位英文名称
- 点位类型
- 行政区域

坐标信息：

- 高德展示经度 GCJ-02
- 高德展示纬度 GCJ-02
- 坐标转换方式
- 坐标转换时间
- 原始 CGCS2000 坐标审计信息入口

业务信息：

- 信号机厂商
- 感知点位类型
- 是否自适应点位
- 是否可变车道路口
- 可变车道数量
- 可变车道方向

### 6.3 嵌入式地图

地图展示：

- 当前点位：高亮显示。
- 周边点位：显示最近 4-5 个点位。
- 不同行政区域点位用不同颜色显示，当前点位在区域色基础上加高亮描边。
- 地图自动缩放到可同时看到当前点位和周边点位。
- 显示比例尺和缩放控件。

周边点位列表：

- 排名
- NodeID
- 点位名称
- 点位类型
- 行政区域
- 距离米

交互规则：

- 点击周边点位可打开摘要。
- 摘要中提供跳转到对应点位编辑页。
- 点击 `重新转换坐标` 后刷新 GCJ-02 坐标和地图。

## 9. 导入流程原型

### 9.1 点位表导入

```text
上传 Excel → 识别字段 → 校验 NodeID/字典/坐标 → 坐标转换 → 预览结果 → 确认导入 → 导入报告
```

导入报告展示：

- 总行数
- 成功行数
- 失败行数
- 重复 NodeID
- 坐标异常
- 字典异常

### 9.2 设备安装表导入

```text
上传 Excel → 识别 Sheet1/未匹配 → 按 NodeID 匹配点位 → 校验数量字段 → 保存单据索引 → 生成未匹配池 → 导入报告
```

导入报告展示：

- Sheet1 总行数
- 匹配成功行数
- 匹配失败行数
- 未匹配工作表行数
- 开箱验收问题数
- 数量异常数
