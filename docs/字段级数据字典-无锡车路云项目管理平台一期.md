# 字段级数据字典：无锡车路云项目管理平台一期

## 1. 数据字典说明

本文档定义一期核心数据对象的字段结构，重点覆盖点位管理、点位地图、设备安装位置、设备未匹配池、一点一档和坐标转换。

本次补充合同管理数据对象，覆盖合同流、分项包、合同明细、设备级合同匹配、合同履约和付款节点。

字段类型采用产品设计口径，研发可按实际数据库选型映射。

## 5. 点位周边关系缓存表 `site_neighbor_cache`

本表不允许人工维护，由系统自动计算生成。触发时机包括：点位导入、点位坐标变更、坐标重新转换、点位删除、批量刷新周边点位缓存。

| 字段名 | 中文名 | 类型 | 必填 | 说明 |
|---|---|---:|---|---|
| id | 系统主键 | bigint | 是 |  |
| site_id | 当前点位 ID | bigint | 是 |  |
| neighbor_site_id | 周边点位 ID | bigint | 是 |  |
| distance_meters | 距离米 | decimal(12,2) | 是 | 基于 GCJ-02 计算 |
| rank_no | 排名 | int | 是 | 1-5 |
| calculated_at | 计算时间 | datetime | 是 |  |

唯一约束：

- `site_id + neighbor_site_id` 唯一。

自动计算规则：

1. 仅使用 `coordinate_convert_status=success` 的点位参与计算。
2. 使用 GCJ-02 坐标计算球面距离。
3. 排除当前点位自身。
4. 每个点位默认计算最近 5 个周边点位。
5. 若当前点位所在行政区域不足 5 个周边点位，则在全项目范围内补足。
6. 计算结果按距离升序写入 `rank_no`。

| 接线表 | 施工单位、信号机品牌、信号灯位置、信号灯类别、数量、接线位置、端子号 | signal_wiring_form、signal_wiring_item |
| 点位照片 | 点位全景、设备照片、封面图 | site_archive_photo、document_file |
| 设备清单 | 按设备类型逐项展示安装位置、物料编码、物料名称、供应商、合同、数量、状态 | device_installation_item |

| 运维记录 | 工单、维修、更换、巡检 | work_order、inspection_record |

自动生成规则：

1. 系统扫描一路一档资料目录，按文件夹名称生成 `site_archive_folder`。
2. 通过文件夹名称、验收单点位名称、接线表点位名称与 `site.site_name` 进行标准化匹配。
3. 匹配成功后自动解析 `验收单.docx`、`接线表.xls` 和点位照片。
4. 解析失败的字段保留原始文件，并在 `parse_message` 中记录。
5. 一点一档页面实时聚合结构化数据和附件资料，不要求人工重复录入。
